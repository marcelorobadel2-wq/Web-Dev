Installation & Setup

1. Unzip the Project

Extract the downloaded zip file to a folder on your computer.

2. Install Dependencies

Open your terminal in the project folder and run:

npm install

3. Start Development Server

Run the local development server to view the site:

npm run dev
The site will be available at http://localhost:5173

Customization

1 Changing Colors

We use a simplified Tailwind configuration injected in index.html. Locate the tailwind.config script tag: 

colors: {
  brand: {
    500: '#0ea5e9', // Change your primary color here
    accent: '#8b5cf6', // Change your secondary color here
  }
}

Deployment

Production Build

To create a production-ready version of your site, run:

npm run build

This will generate a dist/ folder containing all your optimized assets (HTML, CSS, JS). You can upload this folder to any hosting provider.

Netlify (Recommended)

Drag and drop your dist folder to Netlify Drop for instant hosting.

Vercel

Connect your GitHub repository and Vercel will auto-deploy on every push.
